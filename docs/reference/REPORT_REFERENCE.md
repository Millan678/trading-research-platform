# Report Reference

Report formats generated by each unification phase. All reports are produced in both JSON and Markdown.

---

## Report Formats

Each report is generated as:
- `<slug>.json` — Machine-readable JSON
- `<slug>.md` — Human-readable Markdown

## JSON Format

```json
{
  "report_id": "REPORT_ID",
  "advisory_only": true,
  "data": { ... },
  "integrity_hash": "sha256:<hex>",
  "generated_at": "ISO-8601"
}
```

## Markdown Format

Standard semantic markdown with:
- Title header
- Summary table
- Detailed sections
- Advisory-only footer

**All reports are advisory-only.**
